# Rule-Based AI Chatbot 🤖

## Project Overview

This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot responds to predefined user inputs using decision-making logic and runs continuously until the user enters an exit command.

This project was created as part of the DecodeLabs Artificial Intelligence Industrial Training Program.

## Features

* Handles greetings and common questions
* Uses predefined responses
* Runs in a continuous loop
* Handles uppercase and lowercase input
* Provides a default response for unknown inputs
* Exit command using `bye`

## Technologies Used

* Python 3

## How It Works

The chatbot uses:

* `while True` loop for continuous interaction
* Dictionary-based response storage
* Conditional statements for decision making
* String methods such as `.lower()` and `.strip()`

## Installation & Usage

1. Clone the repository:

```bash
git clone https://github.com/oishi-sarker/Task-1-OishiSarker.git
```

2. Navigate to the project folder:

```bash
cd decodelabs-project1
```

3. Run the chatbot:
